QUERYING W.H.O MENTAL HEALTH

AND SUICIDE DATA USING SQL

In this SQL Project, I would create and query a dataset of mental health and suicide deaths from 2017-2019. The dataset was provided by the World Health Organization. Some questions answered by this SQL project include; total deaths within the period, total sucide deaths within the period, top 5 countries with the highest death count etc.

ANALYZING NETFLIX DATA

USING TABLEAU

This project was carried out to analyze and visulize findings from the 2020/2021 Netflix datset. Some of the findings visualized in this Tableau dashboard include count of movies and TV shows relased by Country, movies and TV shows distribution, duration of different movies and TV shows, etc. The datset used is a Netflix open dataset found on Kaggle. 2021 WORLD HAPPINESS

REPORT WITH PYTHON

This project is aimed at analyzing the 2021 World Happiness Report using Python. The dataset ranks 149 countries on how happy their citizens perceive themselves to be.The dataset contains a number of factors that may or may not influence the happiness score of countries. In this analysis,I would be exploring the dataset to find the relationship between different variables, I will also be plotting visualizations to show the relationship between the different variables

CYCLISTIC BIKE-SHARE

ANALYSIS WITH R STUDIO

This project was done in pursuance of the Google Analytics Certificate. In this project I will be analyzing data provided by the Cyclistics Marketing team to draw insights that can help them with the next marketing campaign. The dataset contained over 5 million entries of data about Cyclistics bike-share rides from January to December 2021. I used R Studio for this analysis. Some packages I used include; tidyvesre,lubridate,ggplot2 and dplyr.
